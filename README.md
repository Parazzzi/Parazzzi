## About Me

I'm a Unity C# Developer focused on building clear, stable, and enjoyable game experiences across mobile, 2D/3D, and VR/XR projects.

My work centers on gameplay systems, Unity UI implementation, SDK integrations, Firebase/analytics setup, and project stabilization. I like turning prototypes into reliable features, improving existing projects without unnecessary rewrites, and keeping game code readable for the next developer who opens it.

I'm especially interested in mobile games, Meta Quest / XR prototypes, polished interaction flows, and gameplay logic that feels good in the player's hands.

## What I Work On

- Mobile game development

- Unity UI implementation
- VR/XR prototypes
- SDK integrations
- Bug fixing and project stabilization
